# Software defect prediction using Machine Learning

## Project Overview
The aim of this project is to find the best effective Machine Learning algorithm for predicting Software defects. Multiple classification models were implemented and evaluated using standard performance metrics.

## Algorithms used
- Logestic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbour (KNN)
- Naive Bayes
  
## Dataset used
KC1 dataset from NASA software defect datasets

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Results
Random Forest achieved the best performance in terms of Recall and F1-score compared to other models.

##  Reference Research papers
- A Novel Approach to improve Software Defect Prediction Accuracy using Machine Learning (IEEE ACCESS)

## Tools used 
- Python
- skykit-learn

- Pandas
- Google colab
## Conclusion
Random forest is the best suitable Machine learning algorithm for Software Defect Prediction in this project.
